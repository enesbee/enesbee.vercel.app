## 콜센터 (5339번)

```js
console.log(
  '     /~\\\n    ( oo|\n    _\\=/_\n   /  _  \\\n  //|/.\\|\\\\\n ||  \\ /  ||\n============\n|          |\n|          |\n|          |'
```

_예제 출력 1_

```plain
     /~\
    ( oo|
    _\=/_
   /  _  \
  //|/.\|\\
 ||  \ /  ||
============

|          |
|          |
```

```js
  .readFileSync('/dev/stdin')

  .trim()
  .split('\n')
  .map((a) => (result += Number(a)))
console.log(result)
```

_입력_

>
> - i 번째 줄(1 ≤ i ≤ 5)에는 정수 Ai가 적혀있다. 이것은 i번째 게임에서의 JOI군의 점수를 나타낸다.
>   _출력_

> 표준 출력에 JOI군의 총점을 한 줄로 출력하라.
